Herramientas de edición. División de una entidad en múltiples entidades

     Esta herramienta permite segregar una entidad del dibujo en múltiples entidades a partir de la referencia poligonal que se considere para ello. Consta de un único botón dentro de la barra de herramientas con el siguiente aspecto:

     Esta opción estará accesible para todos los usuarios del editor GIS.

     El usuario selecciona la entidad que desea segregar y dividir. Posteriormente dibuja y selecciona las entidades o polígonos de referencia que se van a emplear para realizar la división. A continuación selecciona la herramienta de división de entidades y el  sistema genera las nuevas entidades y lanza una pantalla en la que introducir sus nuevos atributos.

     Para que se habilite el botón que lanza la ejecución de esta herramienta tienen que estar seleccionadas al menos 3 entidades o features (1 entidad a dividir y 2 de referencia), además de tener seleccionada en el árbol de navegación de las capas la capa a la que pertenece la entidad que se quiere dividir.

     Una vez introducidos los datos, el sistema mostrará las nuevas entidades generadas.
