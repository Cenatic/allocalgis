Herramientas de edición. Selección por polígono

  •  Selección por Polígono

     Esta herramienta permite seleccionar todas las entidades que intersecan un polígono dado. Consta de un único botón dentro del panel de herramientas de edición , con el siguiente aspecto:

     El usuario pulsará el botón de esta herramienta de manera que el cursor se mostrará como una herramienta de dibujo. Una vez hecho esto el usuario dibujará en el mapa el polígono que delimitará la zona de selección de las entidades o features.

     Para que se mantengan las selecciones previas habrá que mantener pulsado el botón de Mayúsculas del teclado, en caso contrario se perderá la selección anteriormente realizada. Las entidades o features seleccionadas serán resaltadas en el mapa.

     El método o la manera de dibujar el polígono de selección es idéntico al que se emplea en la herramienta “Dibujar Polígono”, accesible desde el cuadro de edición, con la particularidad de que en este caso no se añadirá ninguna entidad.

Únicamente se seleccionarán las entidades pertenecientes a la capa que se tenga seleccionada.
