Herramientas de edición. Tratamiento de simetrías

      Esta herramienta permite la generación de elementos simétricos a partir de líneas simples o puntos, que se utilizarán como ejes de simetría. Consta de un único botón dentro del panel de herramientas de edición , con el siguiente aspecto:
Esta opción estará accesible para todos los usuarios del editor GIS.

     El usuario dibuja en una capa auxiliar la línea o el punto que desea utilizar como eje de simetría. El usuario selecciona las entidades de las que desea obtener las simetrías. Posteriormente, selecciona la herramienta “Tratado de Simetrías”.
El sistema genera las entidades simétricas a las seleccionadas respecto al eje pintado, y solicita la inserción de los atributos necesarios.

     Si se desea deshacer la operación se podrá hacer pulsando el icono Deshacer . De esta manera cada vez que pulsemos sobre este icono se irán eliminando una a una las distintas simetrías que se hayan realizado.


     Para que la herramienta funcione correctamente y sea capaza de obtener la simetría esperada, habrá que seleccionar antes de presionar sobre el botón de la herramienta una única feature o entidad, además ésta tendrá que ser o bien un único punto o una línea formada por dos puntos. Otra condición necesaria para esta herramienta es que la capa del mapa donde se encuentra la feature sobre la que queremos realizar la simetría debe ser editable. En caso contrario se mostrará un mensaje de aviso indicando la causa del mismo.
