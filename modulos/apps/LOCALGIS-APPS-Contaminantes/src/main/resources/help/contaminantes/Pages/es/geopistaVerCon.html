Actividades contaminantes

      Para acceder a esta pantalla hay que elegir el menú Actividades contaminantes/Ver expedientes actividades contaminantes. Esta pantalla sirve para gestionar las actividades contaminantes existentes en el ayuntamiento. Para buscar una actividad contaminante utilizar la pantalla búsqueda.

Pantalla de actividades contaminantes

     Operaciones

     Las operaciones que se pueden realizar son:.

  • Búsqueda:
      • Busca una dirección dentro del mapa. Esta opción sirve para posicionarse en la zona elegida dentro del municipio. Para buscar una actividad contaminante utilizar la pantalla de búsqueda.
  • Editar:
      • Esta opción permite editar los datos de un actividad previamente seleccionada. Para seleccionar una actividad se puede hacer o bien seleccionando una actividad del mapa o buscando la en la pantalla de búsqueda. Después de modificar los datos de una actividad hay que pulsar el botón Salvar para que estos sean salvados en el sistema.
  • Borrar:
      • Elimina una actividad del sistema. Para ello hay que seleccionar previamente la actividad contaminante que se desea borrar.
  • Crear:
      • Para crear una actividad hay que seleccionar una herramienta de dibujo de la caja de edición del mapa y posicionar la actividad que se desee crear, rellenar los datos correspondientes a esa actividad y pulsar el botón Aceptar

     Campos

  • Tipo:
      • Para añadir, modificar o eliminar los tipos de actividades contaminantes, utilizando la herramienta de administración editar el dominio TIPO_ACTIVIDAD_CONTAMINANTE de la categoría "Actividades contaminantes".
  • Razón del estudio:
      • Se pueden modificar las razones de estudio mediante la herramienta de administración editando el dominio "RAZON_ESTUDIO" de la categoría "Actividades contaminantes".
  • Posibles infractores:
      • Se puede añadir, editar y borrar posibles infractores a la actividad contaminante.
  • Inspeciones:
      • Dentro de la pantalla de edición de una actividad, existe un listado con las inspecciones de la actividad contaminante que se han realizado. Para añadir, borrar o editar una inspección debe seleccionarse primero el botón de edición de la actividad contaminante, para que se puedan modificar los datos.

Pantalla inspección

     Permisos

     Para poder entrar en esta pantalla el usuario debe poseer el permiso Geopista.Contaminantes.View. Para poder modificar los datos del mapa el usuario debe poseer el permiso Geopista.Contaminantes.Edit.
