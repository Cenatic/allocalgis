Escombreras

      Para acceder a esta pantalla hay que elegir el menú Escombreras/Ver escombreras. Esta pantalla sirve para tener un listado de los vertederos y escombreras del ayuntamiento y llevar la gestión de los mismas. En la parte superior izquierda se muestra un listado con las escombreras. Seleccionando una zona se muestra su situación en el mapa.

Pantalla escombreras

     Operaciones

     Las operaciones que se pueden realizar son:.

  • Búsqueda:
      • Busca una dirección dentro del mapa. Esta opción sirve para posicionarse en la zona elegida dentro del municipio.
  • Editar:
      • Esta opción permite editar los datos de una escombrera previamente seleccionada. Después de modificar los datos de una escombrera hay que pulsar el botón Grabar para que estos sean salvados en el sistema.
  • Borrar:
      • Elimina la escombrera seleccionada del sistema. Para ello hay que seleccionar previamente la escombrera que se desea borrar.
  • Crear:
      • Para crear una escombrera hay que seleccionar un herramienta de dibujo de la caja de edición del mapa y dibujar la escombrera que se desee crear, rellenar los datos correspondientes a esa escombrera y pulsar el botón Aceptar

    Dentro de la pantalla de edición de una escombrera, pulsando en el botón Residuos se pueden ver los residuos que componen la escombrera

    .

Pantalla residuos

     Permisos

     Para poder entrar en esta pantalla el usuario debe poseer el permiso Geopista.Contaminantes.View. Para poder modificar los datos del mapa el usuario debe poseer el permiso Geopista.Contaminantes.Edit.
