Acceso

     Se han definido en el sistema una serie de perfiles de usuario, en función de los cuales, el usuario tendrá restringido el acceso a las opciones de menú.

     Al inicio de la aplicación se muestra la ventana de validación de usuario. En esta ventana, se introducirá la información de validación de usuario y la contraseña.

     Si el nombre de usuario y contraseña se corresponden con la de un usuario con permisos para entrar en la aplicación de gestión medioambiental se le permitirá seguir ejecutando la aplicación, si la autenticación y la autorización son negativas, es decir, no existe un usuario con el nombre y contraseña introducidos con los permisos adecuados, el usuario no podrá acceder a esta aplicación.


     Permisos

     Para que un usuario pueda entrar en la aplicación debe de poseer el permiso Geopista.Contaminantes.Login, si un usuario introduce el login y la password correcta pero no posee este permiso, el sistema le impedirá entrar en la aplicación.
