(C)Citación

     En la pantalla citación se muestran los datos correspondientes a la entidad CI_Citation del estándar que proporciona una referencia a los datos que se están describiendo. Esta pantalla es llamada cada vez que se quiere dar el nombre de cualquier documento, objeto o fuente documental relacionada con el conjunto de datos. Los campos mostrados son:

  • Titulo:
      • Nombre por el que se conoce el recurso mencionado.
  • Fechas:
      • Fechas de referencia para el recurso mencionado. Para añadir o borrar una fecha hay que pulsar el botón derecho del botón y elegir la opción deseada, si se elige añadir se mostrará la pantalla fecha donde se pueden crear nuevas fechas.
