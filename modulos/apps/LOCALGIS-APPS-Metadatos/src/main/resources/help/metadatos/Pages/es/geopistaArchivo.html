Menú archivo

      En este menú se puede abrir y cerrar la sesión contra el sistema. El usuario podrá trabajar sin tener abierta una sesión pero no tendrá acceso a aquellas operaciones en las que se extrae o se introduce información en el sistema, ya que para esto el usuario debe tener conexión, haber sido autenticado y debe poseer los permisos adecuados.

     Abrir sesión

      Pulsando esta opción del menú archivo, si el usuario no ha abierto una sesión previamente en el sistema, aparece la ventana de acceso al sistema. Si el usuario es correcto se abre una sesión, es entonces cuando el usuario puede interactuar remotamente con el sistema.

     Cerrar sesión

      Cierra una sesión abierta previamente. A partir de este momento el usuario sólo puede trabajar en local.
