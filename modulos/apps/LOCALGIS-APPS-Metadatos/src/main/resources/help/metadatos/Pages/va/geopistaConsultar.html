Consultas

     Para acceder a esta pantalla hay que elegir el menú Gestión metadatos/Consultar metadatos. Esta pantalla permite ver el listado de registros de metadatos almacenados en el sistema, sino se ha realizado previamente el inicio de sesión (ver abrir sesión) no se podrá hacer ninguna búsqueda. Esta lista muestra el identificador(ID) del metadato en el sistema, el título del metadato, el responsable, la fecha y el resumen. El listado se puede ordenar por cualquiera de las cuatro pulsando en la cabecera de cada una. También se pueden realizar una serie de acciones sobre un metadato seleccionándolo. .

     Búsquedas

     Se puede filtrar el listado de metadatos dependiendo de una serie de campos que veremos a continuación, para ellos rellenaremos el filtro elegido y pulsaremos el botón Buscar. Si añadimos mas de un criterio de busqueda se concatenaran. Para volver a ver la lista completa pulsar el botón Todos. Los criterios de búsqueda son:

  • Capa:
      • Nombre de la capa de geopista a la que esta asociado el metadato.
  • Título:
      • Título del metadato. Se buscaran todos los títulos que contengan la secuencia de caracteres introducida en este campo.
  • Identificador:
      • Identificador interno asociado al metadato. Este identificador aparece en la parte inferior en la pantalla edición del metadato.
  • Fecha (desde/hasta):
      • Periodo de tiempo en el que esta comprendida la fecha del metadato
  • Responsable:
      • Persona u organización responsable del metadato. Pulsando el botón adjunto a este campo se mostrará la pantalla de contactos donde se puede seleccionar un responsable

     Operaciones

     Una vez buscado el conjunto de metadatos deseado se pueden realizar una serie de acciones, para ello es indispensable que se seleccione de la lista el metadato que se desea manejar. Estas acciones son:

  • Importar XML:
      • Esta función permite seleccionar el fichero que se quiere importar. El metadato almacenado en el fichero se considerará como un metadato nuevo dentro del sistema.
  • Exportar XML:
      • Esta función exporta a formato XML un metadato previamente seleccionado. Al exportar este metadato no se guardan los identificadores internos propios del sistema, por lo que si se vuelve a importar el fichero generado, se considerará que se esta introduciendo un metadato nuevo.
      A la espera de que exista un estándar para la formación de este XML (tanto para la importación como para la exportación) el formato elegido es compatible con la herramienta para la gestión de metadatos del consejo superior geográfico CatMDEdit 3.5 que esta basado en el estándar ISO19155.
  • Salvar a fichero:
      • Esta opción salva en un fichero XML (con formato propietario) todos los campos de un metadato. Se salvan todos los identificadores internos, por lo que se puede modificar el fichero generado y luego cargar el fichero modificando el metadato en el sistema.
  • Cargar fichero:
      • Carga un fichero xml generado previamente con la opción anterior en la pantalla de edición de metadatos. El metadato no se guardará en el sistema hasta que no se pulse el botón salvar de esta pantalla.
  • Mostrar:
      • Muestra un metadato seleccionado en la pantalla de edición de metadatos.
  • Eliminar:
      • Elimina un metadato del sistema.
  • Nuevo:
      • Muestra la pantalla de edición de metadatos en modo creación.
  • Modificar todos:
      • Modifica en todos los metadatos del sistema ciertos valores.

     Permisos

     Para modificar los datos de los metadatos (opción eliminar) el usuario debe estar autenticado y poseer el permiso Geopista.Metadatos.Edit. Para poder buscar y ver la lista de metadatos el usuario debe poseer el permiso Geopista.Metadatos.View y si el metadato pertenece a una capa debe tener permisos para editar dicha capa Geopista.Layer.Escribir.
