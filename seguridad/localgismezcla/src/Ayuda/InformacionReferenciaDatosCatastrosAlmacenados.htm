Paso 2. Almacenar Datos

En el paso 2, se visualizará el fichero gráfico (shp) que se ha seleccionado correspondiente a las parcelas, y se cargará una capa auxiliar, para su posterior almacenamiento en la base de datos con las herramientas de LocalGIS.

Como se ve en la figura aparecerá cargado el mapa con las parcelas, en este caso, y justo debajo una ventana de información, indicando:

  • Registros procesados que se almacenan en la base de datos.
  • Tiempo de finalización del proceso de importación.
