Importar topónimos

Para importar un fichero de topónimos diríjase al módulo denominado "Información de referencia" y seleccione la opción "Importar Topónimos." El fichero de topónimos debe ser de tipo "shapefile (*.shp)" cuya geometría debe ser de puntos.

Automáticamente, se cargará el mapa de Información de Referencia sobre el cual se van a importar los topónimos.

 

Elija el fichero que desea importar pulsando sobre el botón de cargar fichero, representado con el icono de una una carpeta abierta.

Una vez hecho esto, el sistema le indicará que esta operación puede tardar unos minutos. Por favor, tenga paciencia y espere a que finalice el proceso. En caso de error, la causa más probable seguramente sea que el fichero a importar no cumple con el esquema definido para un fichero de topónimos.
