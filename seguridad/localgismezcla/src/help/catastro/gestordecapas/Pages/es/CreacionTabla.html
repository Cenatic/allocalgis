Creación de una nueva tabla de Base de Datos

     La creación de una nueva tabla en la base de datos, supondrá a su vez la generación automática de una nueva tabla de sistema. El proceso a seguir es el que se expone a continuación:

  • Pinchar sobre el botón Añadir, situado en la zona inferior izquierda, bajo la lista de tablas. Se mostrará un cuadro de diálogo en el que habrá que introducir el nombre de la tabla y el tipo de datos que contendrá (este tipo de datos se utilizará básicamente para la creación de la tabla de sistema correspondiente), pudiendo tratarse de datos geométricos o alfanuméricos.

  • Una vez rellenados los campos, se pulsa Aceptar. Si los campos no se han rellenado, se lanza un mensaje de error indicándolo.
  • Si la tabla se crea de modo satisfactorio, se actualiza la lista de tablas de la izquierda. En caso contrario, se muestra un mensaje de error. Para ver la causa, se pulsa el botón Mostrar Detalles, que amplía la información, indicando el mensaje de error que devuelve la base de datos al intentar realizar la operación.

  • NOTA: Dado que se entiende que las tablas de nueva creación se van a utilizar para la generación de nuevas capas, se introducen automáticamente una serie de columnas en base de datos y en sistema:
    • Columna num érica de 8 posiciones, de valores no nulos y únicos, llamada id.
    • Columna numérica de 5 posiciones, de valores no nulos, llamada id_municipio.
    • Además, en caso de haber creado una tabla geométrica, se añadirá también una columna de tipo public.geometry, llamada GEOMETRY.
