Eliminación de una tabla de Base de Datos

     La eliminación de una tabla en la base de datos, supondrá a su vez la eliminación de la tabla de sistema asociada (en caso de que ésta exista, para lo cual deben coincidir los nombres de ambas).

     El proceso a seguir es el que se expone a continuación:

  • Pinchar sobre el botón Eliminar, situado en la zona inferior izquierda, bajo la lista de tablas. O bien pulsar el botón Supr después de haber seleccionado la tabla a eliminar.
  • Se mostrará un cuadro de aviso indicando que la tabla va a ser borrada, implicando este hecho la eliminación de la tabla de sistema asociada, en caso de que la tenga.

 ADVERTENCIA: Si el usuario prosigue con la eliminación, la tabla de base de datos y de sistema se borrarán, así como sus columnas de base de datos y de sistema asociadas, impidiendo cualquier recuperación futura de las mismas y de los datos que contenían.

  • Una vez finalizado el proceso, la tabla borrada ya no se mostrará en la lista.
