Herramientas de edición. Fusión de elementos

     Esta herramienta permite fusionar las entidades seleccionadas del dibujo. Consta de un único botón dentro de la barra de herramientas, con el siguiente aspecto: . Al pulsarlo, aparece la siguiente interfaz:


     Esta interfaz consta de los siguientes elementos:

  • Un desplegable donde se recogen todas las capas del mapa actual, y en el que el usuario deberá escoger aquélla en la que desee crear la nueva entidad (procedente de la fusión de las otras)
  • Un botón de Aceptar, que permite iniciar el proceso de fusión
  • Un botón de Cancelar, que permite regresar a la pantalla de procedencia sin realizar ninguna acción.

     Esta opción estará accesible para todos los usuarios del editor GIS.

     El usuario escoge el conjunto de entidades que desea fusionar dentro del mapa y selecciona la herramienta de fusión de la barra de herramientas. Después el sistema muestra la interfaz en la que el usuario debe indicar la capa en la que se va a crear la nueva entidad, una vez seleccionada la capa pulsará el botón de Aceptar y el sistema fusiona las entidades seleccionadas, creando una nueva.
