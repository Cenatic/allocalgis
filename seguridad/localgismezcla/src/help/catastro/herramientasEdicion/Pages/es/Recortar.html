Herramientas de edición. Recortar los elementos del dibujo

     Esta herramienta permite recortar entidades de un dibujo respecto a una geometría indicada. Consta de un único botón dentro del panel de herramientas de edición , con el siguiente icono:
 
     Esta opción estará accesible para todos los usuarios del editor GIS.

     El usuario seleccionará una entidad de referencia, que servirá como plantilla para la realización del corte. Esta entidad puede estar presente en el mapa o puede dibujarse en una capa auxiliar. A continuación, el usuario elige la herramienta de recorte de elementos,  cambiando la forma del cursor. Posteriormente el usuario pasará a seleccionar las entidades que desea recortar.

     El sistema realiza el recorte de la entidad, eliminando todas aquellas zonas que se queden dentro de la entidad de referencia.
Para que la herramienta funcione correctamente y realice el recorte de las entidades seleccionas, deben cumplirse dos condiciones:

  • Hay que tener seleccionada una única entidad de referencia, en caso contrario se mostrará un mensaje de aviso y no se realizará la acción.
  • Las capas donde se encuentren las entidades sobre las que se pretende realizar el recorte deben ser editables, en caso contrario se mostrará un mensaje de aviso y no se realizará la acción.
