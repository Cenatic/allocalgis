Búsqueda de Expedientes

     La interfaz inicialmente mostrará una lista con todos los expedientes que se encuentran pendientes de resolución. Llegados a este punto, el usuario podrá modificar los resultados de dicha lista de expedientes filtrando a través de los diferentes criterios de búsqueda que ofrece la interfaz.

     De esta forma, podrá conocer los expedientes que se encuentran en un estado especifico, o que se han registrado en un periodo de tiempo determinado o inclusive que ya han sido cerrados y resueltos. También desde esta misma pantalla, el usuario podrá crear un nuevo expediente desde cero, pulsado en la opción "Nuevo" e introduciendo los datos correspondientes.

     Si desea abrir un expediente ya registrado para modificar su información, editar su información catastral, asociar parcelas o bienes inmuebles o ver su estado, debera seleccionar el expediente que desee y pulsar el boton "Aceptar", con lo que se abrira la pantalla de Gestión de Expediente, donde podra realizar todas estas acciones.

     En la pantalla se muestran diferentes campos para rellenar, dependiendo del filtrado que se deseará ralizar.

  • Los campos para el filtrado son:
    • Número de Expediente : Puede realizar el filtrado por el numero de expediente o por parte del mismo.
    • Año del expediente : El año del expediente, en formato AAAA.
    • Codigo de la entidad origen : Puede seleccionar uno de la lista o "seleccionar uno", que no causará filtrado para este campo.
    • Tipo de expediente : Puede seleccionar uno de la lista o "seleccionar uno", que no causará filtrado para este campo.
    • Estado de Expediente : Puede seleccionar uno de la lista o "seleccionar uno", que no causará filtrado para este campo.
    • Usuario : Puede seleccionar uno de la lista o "seleccionar uno", que no causará filtrado para este campo.
    • Fecha de registro : Puede seleccionar un rango de fechas, desde una fecha o hasta una fecha para realizar el filtrado.
    • Parcela Catastral 1 : Puede introducir una parcela catastral o parte de ella.
    • Parcela Catastral 2 : Puede introducir una parcela catastral o parte de ella.
    • Nif Presentador : Puede introducir el nif completo o parte del mismo.
    • Nombre Presentador : Puede introducir el nombre completo o parte de él.

     El filtrado se realizará sobre todos los campos rellenados y las lista que no contengan "seleccionar uno". Por ejemplo, si se deseara realizar un filtrado sobre el "Nif del Presentador" y el "Estado del Expediente", se deberá selecionar un estado y rellenar el campo del Nif. Si posteriormente solo se quisiera filtrar por estados, se debería borrar los datos del "Nif del Presentador".

     Los resultados del filtrado se mostrarán en la tabla de la pantalla. El campo "Parcela Catastral" de la tabla es una lista con todas las parcelas o bienes inmuebles asociados al expediente.
