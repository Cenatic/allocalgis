Arbolado

      Para acceder a esta pantalla hay que elegir el menú Arbolado/Ver arbolado. Esta pantalla sirve para tener un listado de las zonas arboladas del ayuntamiento y llevar la gestión de las mismas. En la parte superior izquierda se muestra un listado con las zonas existentes. Seleccionando una zona se muestra su situación en el mapa.

Pantalla arbolado

     Operaciones

     Las operaciones que se pueden realizar son:.

  • Búsqueda:
      • Busca una dirección dentro del mapa. Esta opción sirve para posicionarse en la zona elegida dentro del municipio.
  • Editar:
      • Esta opción permite editar los datos de una zona previamente seleccionada. Después de modificar los datos de una zona arbolada hay que pulsar el botón Grabar para que estos sean salvados en el sistema.
  • Borrar:
      • Elimina zona arbolada. Para ello hay que seleccionar previamente la zona que se desea borrar.
  • Crear:
      • Para crear una zona arbolada hay que seleccionar un herramienta de dibujo de la caja de edición del mapa y dibujar la zona arbolada que se desee crear, rellenar los datos correspondientes a esa zona y pulsar el botón Aceptar

     Permisos

     Para poder entrar en esta pantalla el usuario debe poseer el permiso Geopista.Contaminantes.View. Para poder modificar los datos del mapa el usuario debe poseer el permiso Geopista.Contaminantes.Edit.
