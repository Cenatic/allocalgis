Inspectores

      Para acceder a esta pantalla hay que elegir el menú Actividades contaminantes/Inspectores. Esta pantalla sirve para gestionar la información referente a los datos de los inspectores que intervienen en el control de las actividades contaminantes.

     Operaciones

     En esta pantalla se pueden Crear, Modificar y Borrar los datos de los inpsectores de las actividades contaminantes. En la parte izquierda de la pantalla se muestran los inspectores existentes, seleccionando cada nombre vemos la información correspondiente a cada uno de ellos.

  • Añadir:
      • Para crear un nuevo inspector pulsar el botón Añadir, rellenar los datos correspondientes al nuevo inspector y pulsar el botón Salvar.
  • Editar:
      • Seleccionar de la lista el inspector que se quiere modificar. Realizar los cambios oportunos. Pulsar el botón Salvar.
  • Borrar:
      • Elegir un inspector de la lista, pulsar el botón Borrar y confirmar que se quiere eliminar el inspector.

     Permisos

     Para poder entrar en esta pantalla el usuario debe poseer el permiso Geopista.Contaminantes.View. Para añadir, borrar o modificar los datos de un inspector el usuario debe de tener el permiso Geopista.Contaminantes.Edit.
