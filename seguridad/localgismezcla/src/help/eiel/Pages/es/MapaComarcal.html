Mapa Comarcal

     La funcionalidad para este servicio, es similar a la de mapas provinciales. La única diferencia es el ámbito, ya que si en el primero se trata de una provincia, en el segundo es para una comarca.

Así pues, una comarca se define como un conjunto de municipios, sin el carácter político y administrativo de una provincia, sino afín a otros parámetros.
