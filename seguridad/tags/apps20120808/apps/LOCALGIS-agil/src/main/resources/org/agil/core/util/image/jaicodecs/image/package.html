JAI Encoders

Este package contiene clases para cargar imagenes en diferentes formatos:
TIFF
GIF
PNG
JPEG
etc

Las clases proceden del package javax.media.jai.codecs, cuyos fuentes fueron distribuidos por Sun, para poder cargar imagenes sin tener que utilizar la distribucion completa del JAI.

Con el JDK 1.4 se ha introducido una nueva arquitectura para la manipulacion de imagenes, el API javax.imageio, que contempla la carga de diferentes formatos graficos añadiendo plugins (en forma de Jars). No obstante, el API imageio actualmente solo contempla los formatos PNG,GIF y JPEG para lectura, y PNG,JPEG para escritura. Conforme se vayan añadiendo nuevos plugins a imageio, se iran sustituyendo las clases de este package.
