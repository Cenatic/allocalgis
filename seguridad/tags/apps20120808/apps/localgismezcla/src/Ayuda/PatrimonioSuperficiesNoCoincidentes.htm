LISTADO DE SUPERFICIES NO COINCIDENTES

Mediante este listado se mostrarán las parcelas cuyas superficies, la calculada a través de la geometría y la que el usuario introduce alfanuméricamente, sean distintas.

Esto se va a realizar a través de un asistente:

En la primera ventana se cargará un plano de referencia que contendrá por lo menos la capa de parcelas.

En esta interface, aparece una barra de menús, con la que se podrá realizar las siguientes tareas:

Pinchando y arrastrando sobre el mapa se provoca un zoom a la zona seleccionada.

Haciendo clic con el botón derecho sobre el mapa se provoca un alejamiento

Herramienta para desplazar el plano por la pantalla

Herramienta para seleccionar un elemento del plano

Al usar la herramienta de selección sobre una parcela, se generará dos consultas:

  • Una a la base de datos para obtener su superficie alfanumérica
  • Otra consulta sobre la geometría, para calcular la superficie de la parcela seleccionada.

De esta forma se podrán consultar on-line las superficies de las parcelas que se deseen.

Cuando se pulse el botón de Finalizar se generará un listado con las parcelas en las que no coinciden ambas superficies.
