IMPORTAR FICHERO INFORMÁTICO DE CATASTRO URBANA

Selección del fichero

 

Esta pantalla, le permite al usuario importar el FINURB (Fichero Informático de Catastro Urbana), proporcionado por la Dirección General de Catastro que desea incorporar en la base de datos del sistema.

El usuario podrá seleccionar archivos de tipo ".txt", para proceder a la importación de los datos de los Inmuebles Urbanos.

El usuario deberá tener el permiso de "Importar FINURB", para poder seleccionar el fichero y continuar con la importación. De lo contrario el botón de Abrir aparecerá desactivado, y el usuario no podrá realizar ninguna acción en esta pantalla.

Una vez seleccionado el fichero a importar, el sistema validará la información y la intentará cargar en la base de datos del sistema.

Si el sistema encuentra errores o discrepancias en la información, imprime un mensaje de error y no importará ninguna información para mantener la integridad del sistema.

 

Si por el contrario, no se encuentra ningún error se guardará toda la información en la base de datos e imprimirá un mensaje diciendo que la operación se ha realizado con éxito.
