Importar Datos de la D.G. de Catastro

Mediante esta opción se importarán los datos de las parcelas provenientes de un fichero en formato shapefile (.shp), dentro de la base de datos en la tabla parcelas.

Se lanzará un asistente con dos pasos:

En el primero se pedirá unos datos generales

En el segundo se visualizarán los datos

 

Paso 1. Selección del fichero.

Se rellenarán los siguientes campos:

Fuente: Donde proviene la información

Organismo:Organismo que posee la información

Persona: Persona que realiza la importación

Fecha: Fecha de la importación

Tipo : Tipo del fichero

Se deberá seleccionar el fichero de parcelas, que indica el cuadro combinado, haciendo clic sobre el botón

Aparecerá un cuadro de diálogo en el que se buscará el fichero por el equipo del cliente.

Se pulsará el botón abrir y se comenzará un proceso de validación del fichero. Las validaciones que se preveen afectan a:

Campos. Nombres de los campos que se esperan que vengan en el fichero

Valores. Se evaluarán los valores de los campos principales, revisando especialmente valores nulos y valores que no sean del tipo adecuado.

Una vez comprobadas las verificaciones se mostrarán un mensaje donde se indicará si el fichero ha sido correcto y se puede continuar con el siguiente paso, o bien si se el proceso de validación ha encontrado errores y no dejará continuar.

Si el fichero es correcto saldrá:

 

Paso 2. Almacenar Datos

En el paso 2, se visualizará el fichero gráfico (shp) que se ha seleccionado correspondiente a las parcelas, y se cargará una capa auxiliar, para su posterior almacenamiento en la base de datos con las herramientas de LocalGIS.

Como se ve en la figura aparecerá cargado el mapa con las parcelas, en este caso, y justo debajo una ventana de información, indicando:

Registros procesados que se almacenan en la base de datos.

Tiempo de finalización del proceso de importación.
