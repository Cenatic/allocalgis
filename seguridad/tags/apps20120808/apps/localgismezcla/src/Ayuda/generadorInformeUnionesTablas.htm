Uniones entre Tablas

Las uniones entre tablas son necesarias para que la información que salga sea correcta.

En los cuadros combinados correspondientes a las Tablas aparecerán las tablas que tienen campos en el informe.

Hay que tener en cuenta que esta interface requiere que el usuario conozca el modelo de datos para poder realizar las uniones entre las tablas.

Se seleccionará un campo de la tabla de origen con un campo de la tabla de destino y se pulsará el botón

A medida que se realicen las uniones estas aparecerán en la tabla , si se desea eliminar la unión se hará clic sobre el botón

En esta ventana también se podrá realizar un grupo con algún campo que aparezca en el informe:

Se activará mediante la casilla de verificación Agrupar

Se seleccionará el campo Agrupar por, y se seleccionará algún campo que esté presente en el informe

La etiqueta que aparezca será la que se ponga en el campo Título
