Forma de utilización

     Se exponen a continuación los pasos a seguir para realizar las distintas operaciones que permite la herramienta.

  • Crear una nueva familia o categoría de capas

  • Añadir una o más capas a una familia de capas

  • Modificar el nombre de una familia de capas

  • Eliminar una capa de una layerfamily

  • Modificar el orden de las capas dentro de una familia de capas

  • Eliminar una familia de capas existente
