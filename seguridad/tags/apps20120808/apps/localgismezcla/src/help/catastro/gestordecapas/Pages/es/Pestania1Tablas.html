Pestaña 1: Tablas

     La pestaña Tablas se utiliza para gestionar las distintas tablas de base de datos, así como para controlar su relación con las de sistema. El aspecto de la pantalla es el que se muestra en la siguiente figura:

     Se observa que la pantalla está dividida en tres regiones diferenciadas:

  • En la parte izquierda aparece una lista con todas las tablas existentes en la base de datos, independientemente de que estén reflejadas en tablas de sistema o no. Debajo de la lista, se muestran tres botones que permiten realizar diferentes acciones sobre las tablas: añadir, editar y eliminar.

  • En la parte derecha aparece una tabla en la que se cargarán las columnas de la tabla que se haya seleccionado en la lista de la izquierda y una botonera con las diferentes acciones que se pueden realizar sobre las columnas: añadir, editar, eliminar y crear columna de sistema. La tabla tiene el siguiente aspecto:

     En la tabla aparecen dos columnas:

  • En la primera aparecen los nombres de las columnas de base de datos.
  • En la segunda se muestran los nombres de las columnas de sistema asociadas a las de base de datos. Sólo aparecerá columna de sistema asociada si se encuentra alguna cuyo nombre sea el mismo que una columna de base de datos (sin diferenciar mayúsculas y minúsculas).
