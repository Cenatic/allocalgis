Crear

     Esta opción esta en el menú Gestión metadatos/Crear metadato. La pantalla que se muestra es la de edición de metadatos, posibilitando la creación de un nuevo metadato en el sistema.

     Operaciones

     En la pantalla de edición de metadatos las operaciones que se pueden realizar son las siguientes:

  • Salvar:
      • Graba en el sistema el metadato que se muestra en la pantalla. Para realizar esta opción es imprescindible que el usuario haya abierto un sesión en el sistema (ver abrir sesión).
  • Eliminar:
      • Elimina un metadato del sistema.
  • Salvar a fichero:
      • Esta opción salva en un fichero XML (con formato propietario) todos los campos de un metadato. Se salvan todos los identificadores internos, por lo que se puede modificar el fichero generado y luego cargar el fichero modificando el metadato en el sistema.
  • Cargar fichero:
      • Carga un fichero xml generado previamente con la opción anterior en la pantalla de edición de metadatos. El metadato no se guardará en el sistema hasta que no se pulse el botón salvar de esta pantalla.
  • Imprimir:
      • Imprime el metadato mostrado en pantalla.
  • Validar:
      • Comprueba que los datos son correctos.

     Permisos

     Para modificar los datos del metadato en aquellas operaciones que interactúan con el sistema el usuario debe poseer el permiso Geopista.Metadatos.Edit y si el metadato pertenece a una capa debe tener permisos para editar dicha capa Geopista.Layer.Escribir.
