GENERADOR DE CONSULTAS.

Mediante esta interface se guardarán consultas para poder usar posteriormente en filtros.

 

 

En la parte superior aparecerán en una lista las consultas que están almacenadas

Haciendo clic sobre una consulta de la lista anterior se rellenará toda la interface con la información guardada.

 

Acciones sobre las consultas.

Limpiar la ventana todos los restos de otras consultas y así se podrá iniciar mejor una nueva consulta.

Una vez limpiado la ventana y rellenada la interface de la consulta, se pulsará el botón nuevo para guardarla.

Elimina una consulta de la lista

La consulta que se ha modificado en la interface la almacena en disco.

 

Para cada consulta será obligatorio rellenar el Nombre de la consulta que se le dará, se establecen verificaciones para no nombrar a dos consultas con el mismo nombre.

 

Para el ejemplo se ha cargado de la base de datos, de la tabla provincias los campos correspondientes, el nombre de la tabla se pasa como parámetro.

Campo: Nombre del campo para establecer la condición

Operador: Cada uno de los operadores establecidos para formar la condición

Valor : Valor con el que se va a comprar.

Unión: Será un enlace con el valor anterior, en condiciones múltiples, estos serán Y o O lógicas.

Negar condición: Se usará para negar condiciones.

 

Hay que tener en cuenta que en una condición sencilla de un único valor, no podrá existir ningún enlace con la condición anterior.

Cuando, por el contrario, la condición sea múltiple, a partir de la segunda condición se deberá incluir el enlace con la condición anterior.

 

A medida que se vayan generando las condiciones irán apareciendo en la tabla de la parte inferior de la interface.

 

 

Existen dos botones:

Nos permite añadir la condición que hemos rellenado a la tabla, para posteriormente guardarla con la consulta.

Permite eliminar una fila seleccionada de la tabla anteriormente comentada.
