GENERADOR DE INFORMES.

El generador de informes es la herramienta que posibilita la creación dinámica de informes a gusto del usuario. También podrá seleccionar un informe previamente creado.

El generador de informes consta de un asistente de 7 pasos:

  • Paso 1. Selección /Creación de un informe.
  • Paso 2. Características Generales del Informe.
  • Paso 3. Datos del encabezado del informe.
  • Paso 4. Selección de Campos en el informe
  • Paso 5. Selección de Registros
  • Paso 6. Uniones entre tablas.
  • Paso 7. Finalización del Informe.

 

Selección / Creación de Informe

Esta primera pantalla del asistente, permitirá seleccionar un informe creado previamente con el asistente, o bien se podrá crear un informe nuevo.

Para la selección o creación de un informe se hará clic sobre los botones de opción correspondientes:

Haciendo clic sobre Informe Nuevo se pasará al siguiente paso del generador de informes.

Para usar un informe ya creado se hará clic sobre la otra opción de Selección del Informe. A continuación aparecerán en el cuadro de lista los ficheros que se encuentran ubicados en el equipo.

El nombre de los ficheros se nombra con
  • Nombre del módulo al que pertenece (Planeamiento, Castatro,etc).
  • Nombre del fichero que se le ha dado al crearlo.

Al hacer clic sobre un fichero, se rellenará el Título del Fichero y la Descripción, para una mejor identificación por parte del usuario.

Una vez decidido que acción se va a llevar a cabo (creación o selección) se hará clic sobre el botón Siguiente.
