Editar Red de Abastecimiento

 

Cuando se selecciona esta opción, se abre la pantalla de edición de la red de abastecimiento, donde el usuario puede visualizar o realizar cambios. Esta opción sólo estará habilitada para el usuario con el permiso de "Editor Red de Abastecimiento", en caso contrario no se podrá acceder a esta pantalla.

 

Para esta pantalla, el usuario dispone de un mapa preconfigurado con las capas de Abastecimiento, unas capas básicas de referencia y de un conjunto de herramientas de edición.
