Generar Mapa

Esta opción imprimirá el mapa del módulo de planeamiento que se está visualizando en pantalla.

Se realizará a través de un asistente que consta de los siguientes pasos:

 

Paso 1. Visualización del Mapa

Se ve el mapa cargado con las capas especificadas para este mapa.

En la parte superior se encuentra la barra de herramientas:

Pinchando y arrastrando sobre el mapa se provoca un zoom a la zona seleccionada.

Haciendo clic con el botón derecho sobre el mapa se provoca un alejamiento

Herramienta para desplazar el plano por la pantalla

Haciendo clic sobre esta herramienta se hace un zoom a la extensión del mapa
