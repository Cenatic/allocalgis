IMPORTAR PLANEAMIENTO MUNICIPAL

Selección del fichero

Cuando el usuario selecciona la opción "Importar Planeamiento Municipal" en la pantalla inicial del módulo se abre la pantalla de importación, donde el usuario puede indicar el fichero a importar.

Esta opción sólo está disponible para los usuarios con permisos de "Importar Planeamiento Municipal". Si el usuario no dispone de estos permisos el botón de "Abrir" aparecerá desactivado, y el usuario no podrá continuar en el proceso de importación municipal.

El usuario podrá seleccionar archivos de planeamiento general o de planeamiento de desarrollo que haya dibujado previamente con las aplicaciones para soluciones comerciales.

Solamente se podrán seleccionar shapefiles y jml.

Una vez seleccionado el fichero a importar, el sistema realiza el proceso de validación para comprobar que el fichero es válido.

Si el fichero seleccionado es válido, el usuario podrá seguir con el proceso de importación, pulsando en el botón "Siguiente"

Si el fichero seleccionado no es válido, bien porque no cumple con el esquema especificado, o bien porque aunque cumpla con el esquema hay datos que no son correctos, el usuario no podrá continuar, y el botón de Siguiente aparecerá desactivado, hasta que se seleccione otro fichero correcto o pulse en Cancelar para salir de esta pantalla.
