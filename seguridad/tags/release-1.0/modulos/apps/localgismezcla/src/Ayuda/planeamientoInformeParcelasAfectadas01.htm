Informe de Parcelas Afectadas por Ámbitos de Gestión de Suelo

Este informe recogerá un ámbito de gestión y las parcelas que están afectadas por ese ámbito de gestión del suelo.

Se lanzará la siguiente ventana:

Se verá el plano definido para la opción de ámbitos de gestión.

Esta ventana contiene la siguiente barra de herramientas:

Pinchando y arrastrando sobre el mapa se provoca un zoom a la zona seleccionada.

Haciendo clic con el botón derecho sobre el mapa se provoca un alejamiento

Herramienta para desplazar el plano por la pantalla

Haciendo clic sobre esta herramienta se hace un zoom a la extensión del mapa

Herramienta para seleccionar un elemento del plano

Además contendrá el cuadro combinado, que está rellenado con todos los ámbitos de gestión que existieran en la base de datos.

No se podrá pasar a la visualización del informe, sin que se haya seleccionado un ámbito de gestión. Para ello se podrá realizar de dos formas distintas:

  • Seleccionando un elemento del cuadro combinado, y de forma automática se provocará un zoom en el mapa
  • Haciendo clic con la herramienta de selección del mapa, indicada anteriormente.
