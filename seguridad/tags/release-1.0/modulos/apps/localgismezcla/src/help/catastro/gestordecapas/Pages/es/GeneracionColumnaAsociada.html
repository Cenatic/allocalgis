Generación de columna de sistema asociada a una columna de base de datos

     Para generar una columna de sistema asociada a una de base de datos se pueden seguir dos procedimientos. El primero de ellos consiste en editar la columna de base de datos incluyendo el nombre de la columna en el campo Nombre en el sistema. El segundo, más sencillo, es el que se expone a continuación:

  • Seleccionar la columna de base de datos cuya columna de sistema se desea generar.
  • Pinchar sobre el botón Crear Columnas, situado en la zona superior derecha, junto a la lista de columnas.

  • La columna de sistema se generará automáticamente y la lista de columnas se refrescará.
