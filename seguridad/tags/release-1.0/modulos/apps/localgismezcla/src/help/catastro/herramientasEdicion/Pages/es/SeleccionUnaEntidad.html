Herramientas de edición. Selección de una única entidad.

     Esta herramienta permite seleccionar una única entidad de entre carias existentes. Consta de un único botón dentro del panel de herramientas de edición , con el siguiente icono:

     Funciona del mismo modo que la herramienta de selección, con la particularidad de que sólo se seleccionará una única entidad.

     Esta herramienta se convierte en imprescindible cuando se tienen varias entidades iguales superpuestas y se necesita seleccionar una única entidad de entre todas ellas, ya que algunas de las herramientas de edición requieren tener una única entidad seleccionada.
