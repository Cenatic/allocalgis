Publicador Web

     El publicador Web es la parte de la Guía Urbana que permite inscribir nuevos mapas en el Web. Éstos pueden ser públicos (accesible para todo usuario de Internet) o privados (restringidos a ciertos usuarios propietarios de ciertos permisos).

Para acceder a la Guía Urbana basta con visitar http://localgis.dphuesca.es:8082/localgis-guiaurbana/

El publicador permite la publicación específica de los distintos tipos de mapas, ya sean provinciales, comarcales, municipales, mapas EIEL o temáticos.

En primer lugar, existe un sistema de autenticación de usuarios para que se gestione de forma adecuada los permisos en la publicación de los mapas.

Una vez autenticado, se tiene acceso a aquellos mapas a los que el usuario tiene permiso. Desde este menú, se puede seleccionar el mapa a publicar. Finalmente se mostrará el mapa publicado.

Los mapas provinciales y comarcales se publican de tal forma que al cargar el mapa se centra automáticamente en el término perteneciente a la provincia o a la comarca en particular.
