Visor Web

     El Visor Web es la herramienta que permite a los usuarios visualizar los mapas publicados. Los mapas, como ya se ha citado, serán de ámbito provincial, comarcal o municipal.

Un usuario podrá visualizar un mapa concreto, siempre y cuando tenga el permiso de usuario de LocalGIS correspondiente; excepto en los mapas públicos, los cuales son accesibles para cualquier usuario.

Existe un sistema de alta de usuarios Web para la Guía Urbana con el fin de que pueda publicar sugerencias y anotaciones sobre puntos georreferenciados. Para registrarse, el usuario rellenará un formulario de inscripción y se le remitirá un e-mail de confirmación con la contraseña de acceso. Es importante destacar que los usuarios registrados desde la Guía Urbana son totalmente independientes de los usuarios internos de LocalGIS.

En el visor Web aparece una ventana emergente para añadir nuevos comentarios y visualizar los publicados previamente.
