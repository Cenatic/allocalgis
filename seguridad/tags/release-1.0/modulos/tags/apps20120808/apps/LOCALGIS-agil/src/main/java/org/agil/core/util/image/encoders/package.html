Encoders

Son las clases que convierten una Image en un fichero gráfico.

Este paquete va a cambiar entero. Actualmente codifica las imagenes en ficheros GIF. En breve tiempo se sustituira por el fichero de JAI (es posible en la version 1.4 haya cambios) y se podrán generar imagenes en muchos otros formatos, jpeg, tiff, png, etc,..
