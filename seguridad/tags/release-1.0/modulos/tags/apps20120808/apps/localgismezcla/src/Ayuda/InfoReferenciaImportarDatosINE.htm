IMPORTAR INFORMACIÓN DE REFERENCIA INE

Los datos de importación de referencia del Ine serán:

  • Tramos de vía
  • Tramos vía Ine
  • Números de Policía
  • Tipos Vía Ine
  • Distritos Censales
  • Secciones Censales
  • Subsecciones Censales
  • Entidades Singulares
  • Entidades Colectivas
  • Núcleos y Diseminados

 

Tramos de Vía.

Para importar los tramos de vía, se necesitarán dos ficheros:

Un fichero gráfico, con un listado de los ejes de calle

Un fichero de dbf con las características de las vías.

Una vez pasada la validación se pulsará el botón Siguiente, y se mostrará la información que se va almacenar en la base de datos.

Se muestra la información de registros procesados y la fecha y hora de finalización del proceso.

Tramos vía Ine

Este proceso se rellenará a partir del fichero de texto procesado por el INE (fichero de tramos de vía).

Este proceso no tendrá salida gráfica.

Se pasan los controles de campos y valores indicados anteriormente.

Una vez que el fichero sea correcto se pulsará el botón Siguiente.

 

Presenta la información de resumen, registros procesados y hora de finalización.
