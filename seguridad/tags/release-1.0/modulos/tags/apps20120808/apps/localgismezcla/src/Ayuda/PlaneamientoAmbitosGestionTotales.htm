IMPORTAR AMBITOS DE GESTIÓN DEL SUELO

Selección del fichero

 

Cuando el usuario selecciona la opción "Importar Ambitos de Gestión del Suelo " en la pantalla inicial del módulo se abre la pantalla de importación, donde el usuario puede indicar el fichero a importar.

Esta opción sólo está disponible para los usuarios con permisos de "Importar Ambitos de Gestión del Suelo". Si el usuario no dispone de estos permisos el botón de "Abrir" aparecerá desactivado, y el usuario no podrá continuar en el proceso de importación de ámbitos del suelo.

El sistema comprobará que el archivo a importar contiene ámbitos de gestión del suelo que han sido creados con los plug-in para las aplicaciones comerciales.

Solamente se podrán seleccionar shapefiles y jml.

 

Una vez seleccionado el fichero a importar, el sistema realiza el proceso de validación para comprobar que el fichero es válido.

 

 

Si el fichero seleccionado es válido, el usuario podrá seguir con el proceso de importación, pulsando en el botón "Siguiente"

 

 

Si el fichero seleccionado no es válido, bien porque no cumple con el esquema especificado, o bien porque aunque cumpla con el esquema hay datos que no son correctos, el usuario no podrá continuar, y el botón de Siguiente aparecerá desactivado, hasta que se seleccione otro fichero correcto o pulse en Cancelar para salir de esta pantalla.

 

Comparación de Dominios

 

El sistema compara el rango de valores de los campos principales de las capas de planeamiento con los dominios existentes en el Sistema de Información Territorial. En el caso de existir en el dominio del SIT todo el rango de valores el sistema importa la información sin intervención del usuario. El caso que el nuevo rango de valores incorpore algún valor que no exista en el dominio del SIT el sistema muestra esta pantalla: Comparación de Dominios.

 

El cuadro de diálogo de Comparación de Dominios de planeamiento se divide en tres cuadros de texto, un combo "Dominio" que muestra la capa de dominios y botones de interacción. En el cuadro rotulado como "Valores nuevos" se muestran todos los nuevos valores que el sistema ha detectado automáticamente durante el proceso de importación, la capa a la que hacen referencia se muestra en el combo "Dominio" de la parte superior. En el cuadro "Valores existentes" aparecen los valores del mismo dominio que se encuentran almacenados en el SIT. El cuadro "Lista de Incorporaciones" aparecen los valores que el usuario añade al dominio del SIT.

 

El usuario utilizará el botón "Añadir" para añadir un valor nuevo al SIT. Usará el botón "Vincular" si ha seleccionado un registro del cuadro "Valores nuevos" y otro del "Valores existentes" y el sistema vinculará todos los registros que tengan ese valor del dominio al existente en el SIT como "Valores existentes". El botón "Eliminar" hace que el sistema no importe los registros que tienen ese valor.

 

Para realizar estas operaciones es necesario tener los permisos de "Alta de Dominios Planeamiento Municipal" para añadir, "Vincular Dominios Planeamiento Municipal" para vincular y "Eliminar Dominios Planeamiento Municipal" para eliminar. En caso de no tener estos permisos aparecerán desactivados los botones, por lo que el usuario podrá realizar unas u otras acciones.

 

El sistema validará que el usuario haya ofrecido una solución para todos los valores nuevos encontrados en el proceso previo de importación, de no ser así no podrá proceder a la importación de los registros. Por ello, el botón Siguiente aparece desactivado hasta que no exista ningún valor en la lista de Valores Nuevos.

En el cuadro de texto "Lista de incorporaciones", se mostrará las acciones que el usuario ha optado por realizar.

 

En cualquier momento se puede pulsar sobre el botón "Cancelar" para no modificar estos valores de la base de datos.
