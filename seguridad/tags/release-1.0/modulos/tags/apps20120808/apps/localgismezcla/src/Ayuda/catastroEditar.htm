Editar Catastro

 

Esta opción dispone de un mapa y una tabla de contenidos que mostrará la organización y contenido de dicho mapa y nos permitirá acceder a las distintas capas de información del mapa.

Esta opción sólo estará habilitada para el usuario con el permiso de "Editor de Catastro", en caso contrario no se podrá acceder a esta pantalla.
