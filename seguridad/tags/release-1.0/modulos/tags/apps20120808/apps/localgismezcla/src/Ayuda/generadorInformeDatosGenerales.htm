Características Generales del Informe.

 

Una vez que se decida crear un informe desde cero, se entrará en esta ventana, que permitirá seleccionar los datos y características generales del informe, sin entrar en detalle del contenido del mismo.

 

En esta ventana se podrá elegir:

Campo

Descripción

Módulo

Módulo al que va estar asociado el informe (Planeamiento, Catastro, etc.)

Capa

Se seleccionarán las capas correspondientes a las tablas que existan en la base de datos.

En la ventana de campos, se podrán seleccionar los campos de las capas que se seleccionan en esta ventana.

Nombre Fichero

Nombre físico del informe con el que se almacenará en el disco. Se deberá respetar la nomenclatura de ficheros, sin caracteres especiales.

El nombre del fichero será el nombre del módulo más este nombre.

Orientación

Se seleccionará la orientación que se desea al informe Vertical / Horizontal

Título Fichero

Es un campo descriptivo, no aparecerá en ninguna parte del informe, será sólo para ayudar al usuario a identificarlo en el paso 1 del generador de informes.

Descripción

Igual que el campo Título Fichero

En esta ventana se deberá seleccionar por lo menos :

  • Una capa de donde se sacarán los datos para el informe
  • Titulo Fichero
  • Nombre del Fichero.

Teniendo seleccionados estos elementos se activará el botón de Siguiente
