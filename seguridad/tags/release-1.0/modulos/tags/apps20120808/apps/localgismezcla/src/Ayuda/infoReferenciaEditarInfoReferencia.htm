Editor de Inforeferencia

 Cuando se selecciona esta opción, se abre la pantalla de edición de la información de referencia, donde el usuario puede realizar cambios a la cartografía y datos de la información de referencia. Esta opción sólo estará habilitada para el usuario Editor de Información de Referencia, en caso contrario no se podrá acceder a esta pantalla.

 

Para esta pantalla, el usuario dispone de un mapa preconfigurado con las capas preconfigurado con las capas de la información de referencia y de un conjunto de herramientas de edición.
