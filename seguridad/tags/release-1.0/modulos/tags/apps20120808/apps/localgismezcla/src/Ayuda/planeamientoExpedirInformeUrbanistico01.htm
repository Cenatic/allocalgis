EXPEDIR INFORME URBANÍSTCO

Esta opción generará un informe urbanístico de la parcela que se seleccione sobre el mapa, que aparece en pantalla.

En esta interface aparecen las capas cargadas necesarias para realizar el informe urbanístico.

Además aparece la barra de herramientas que nos va a permitir:

 

Pinchando y arrastrando sobre el mapa se provoca un zoom a la zona seleccionada.

Haciendo clic con el botón derecho sobre el mapa se provoca un alejamiento

Herramienta para desplazar el plano por la pantalla

Haciendo clic sobre esta herramienta se hace un zoom a la extensión del mapa

Herramienta para seleccionar un elemento del plano

Una vez se haga el zoom deseado sobre la zona buscada, se seleccionará la herramienta de selección para identificar la parcela.

Una vez que se identifique una parcela, se rellenarán las casillas de rústica o urbana, dependiendo el tipo de la misma.

 

 

Una vez seleccionada, se pulsará el botón siguiente y se lanzará el Informe Urbanístico.

Hay que tener en cuenta que el informe que se genera y se previsualiza en pantalla se deberá exportar a un formato de archivos PDF, para su posterior impresión.

En la previsualización del informe no se aprecian todos los detalles del mismo, se deberá primero exportar a PDF, como se dijo en el párrafo anterior, y luego imprimirlo.

 

Par exportar un informe a un tipo de fichero se usará la siguiente barra de Herramientas y Menús

Para imprimir un informe se hará de dos formas:

  • Desde el Menú File-Print
  • Con el botón de la barra de herramientas

Para exportar el informe a PDF se hará desde

  • Desde el menú Report-Export
  • Con el botón de la barra de herramientas
