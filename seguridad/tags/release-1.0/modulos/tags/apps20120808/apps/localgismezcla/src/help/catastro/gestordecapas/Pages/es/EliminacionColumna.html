Eliminación de una columna de Base de Datos

     La eliminación de una columna de base de datos, supondrá a su vez la eliminación de la columna de sistema asociada (en caso de que ésta exista).

     El proceso a seguir es el que se expone a continuación:

  • Pinchar sobre el botón Eliminar, situado en la zona superior derecha, junto a la lista de columnas. O bien pulsar el botón Supr después de haber seleccionado la columna a eliminar.  

  • Se mostrará un cuadro de aviso indicando que la columna va a ser borrada, implicando este hecho la eliminación de la columna de sistema asociada.

 Si el usuario prosigue con la eliminación, la columna de base de datos y de sistema se borrarán, impidiendo cualquier recuperación futura de las mismas y de los datos que contenían. Una vez finalizado el proceso, la columna borrada ya no se mostrará en la lista.
