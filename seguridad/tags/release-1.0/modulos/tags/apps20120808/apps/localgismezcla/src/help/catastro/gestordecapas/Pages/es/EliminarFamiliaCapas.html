Eliminar una familia de capas existente

     El proceso de eliminación consiste en los siguientes pasos:

  • Seleccionar del árbol de familias de capas aquélla que se desea borrar. Hay que tener en cuenta que el borrado de una familia de capas no implica el borrado de sus capas.

  • A continuación se pulsa el botón Eliminar. El sistema mostrará una ventana avisando del proceso que se va a realizar.
  • Si el usuario prosigue con el borrado, la familia de capas se borrará del sistema y desaparecerá del árbol de familias de capas.
