Herramientas de edición. Explosión de entidades

     Esta herramienta permite explotar entidades gráficas complejas en los diferentes elementos que la conforman. Consta de un único botón dentro de la barra de herramientas con el siguiente aspecto:

     Esta opción estará accesible para todos los usuarios del editor GIS.

     El usuario selecciona la entidad multi-geometry que desea descomponer en entidades gráficas sencillas utilizando la herramienta de Selección de GeoPISTA y selecciona la herramienta “Explotar Entidades”, una vez realizado esto el sistema divide le entidad gráfica en otras más sencillas.
