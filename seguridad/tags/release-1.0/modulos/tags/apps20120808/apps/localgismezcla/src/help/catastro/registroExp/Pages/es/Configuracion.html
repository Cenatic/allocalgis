Configuración

     La administración de avisos de actualización y envíos periódicos contempla una pantalla de configuración para establecer los parámetros de aviso o alerta al usuario, para que este realice la actualización de los datos con la última situación en catastro o el envío de las últimas situaciones registradas en la entidad colaboradora a la OVC. Además incluye una pantalla de configuración de Modo de trabajo y Convenios con Catastro para dictaminar el flujo a seguir durante el proceso de resolución de expedientes y la información que afectada por los mismos.

     En esta interfaz, se introduce la frecuencia de aviso para realizar dichas operaciones así como se permite activar o desactivar dichos avisos según desee el usuario.

     Los avisos de actualización y envío periódico solo se mostraran a usuarios con perfil de administrador en el sistema y por tanto, serán estos los únicos habilitados para configurar dichos parámetros y para realizar estas operaciones.
