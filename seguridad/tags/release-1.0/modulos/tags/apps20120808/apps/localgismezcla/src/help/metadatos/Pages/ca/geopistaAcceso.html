(C)Acceso

      En esta aplicación el usuario puede trabajar en local o conectado al servidor. Al arrancar aparecerá la pantalla de login donde el usuario tiene la opción de introducir el nombre y la password de una usuario del sistema o trabajar sin conexión pulsando el botón Sin conexión. Con esta segunda opción el usuario tendrá habilitadas todas las funciones de la aplicación independientemente de los permisos que posea.

      Dentro de la aplicación hay funciones en las que es obligatorio que el usuario este conectado al sistema (descarga y carga de información).

      En la parte inferior izquierda de la pantalla principal aparece si el usuario ha abierto una sesión en el sistema o no Sesión abierta/Sesión cerrada. En la parte inferior derecha de la pantalla principal se muestra si el usuario tiene acceso físico al servidor o no Conectado/Desconectado.

      El usuario puede decidir en cada momento conectarse y desconectarse del sistema ultilizando las opciones del Menu de archivo


     Permisos

     Para que un usuario pueda autenticarse en el sistema y trabajar conectado al servidor debe de poseer el permiso Geopista.Metadatos.Login, si un usuario introduce el login y la password correcta pero no posee este permiso, el sistema le impedirá entrar en la aplicación.
