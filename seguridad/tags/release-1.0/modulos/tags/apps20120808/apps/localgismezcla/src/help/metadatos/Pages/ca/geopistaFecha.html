(C)Fecha

     En la pantalla fecha se muestran los datos correspondientes a la entidad CI_Date del estándar que corresponde a la fecha de referencia y el evento usado para describirla. La información que contempla esta ventana es la siguiente:

  • Fecha:
      • Fecha de referencia para el recurso mencionado.
  • Tipo fecha:
      • Evento usado para la fecha de referencia.
